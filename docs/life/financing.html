          # 金融理财

          前言

          别人贪婪我恐惧，别人恐惧我加仓，别人小亏我破产 🤣

          # 我的理财经历

          • 2015 年开始使用余额宝理财
          • 2017 年开始使用度小满理财
          • 2020 年大举买入股票型基金，不到一个月就大赚了 1 万多
          • 2021 年把去年赚的钱全亏进去了，还倒亏 2000 多
          • 2021 年底抛弃股票型基金，转战纯债基金，艰难扭亏为盈
          • 2022 年初债券开始熊市，撤资到银行理财

          # 理财目标

          • 构建低波动资产组合并长期持有
          • 获得长期稳定的现金流

          # 金融常识

          警告

          珍惜生命，远离股市

          # 为什么要建立资产组合

          这年头房住不炒，存款利率逐年降低，只有资产组合（主要包括：债券型基金、货币基金、低利率银行理财）能分散风险，不断增值。

          # 股票型基金 X

          • 股票型基金大致分为：指数型基金、行业基金、混合型基金
          • 无论什么类型的股票型基金波动都非常大
          • 强烈不建议韭菜们买股票型基金

          # 债券型基金 √

          • 债券基金大致分为：纯债基金、一级债基、二级债基
          • 我们需要买的是纯债基金👈
          • 纯债基金分为：长债基金（年化 4-6%）、中短债基金（年化 3-4%）
          • 纯债基金波动很小，可以长期持有
          • 10 年期国债收益率与债券价格负相关
          • 10 年期国债收益率 (opens new window)在 3%以上的时候买入债券基金最好重点

          # 基金买卖费用

          • 除货币基金，所有基金7 天内卖出都收取1.5%的手续费
          • 建议购买 C 类基金，必要时可以随时卖出

          # 资产组合

          1. 债券牛市（10 年期国债收益率在 3%以上，并且不断降低）
            70%长债基金 + 20%中短债基金 + 10%货币基金

          2. 债券熊市（10 年期国债收益率在 3%以下，并且不断拉升）
            80%银行理财 + 20%货币基金

          # 基金推荐自负盈亏

          类型 名称 收益率 APP
          长债基金 招商招旭纯债债券 A 4-6% 支付宝
          短债基金 招商鑫福中短债债券 A 4% 支付宝
          银行理财 中银理财惠享天天进阶版 2.7% 招商银行
          活期理财 朝朝宝 2.5% 招商银行
