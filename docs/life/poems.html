          # 优美诗词

          前言

          生活不止眼前的苟且，还有诗和远方。

          # 题都城南庄 崔护

          去年今日此门中，人面桃花相映红。
          人面不知何处去，桃花依旧笑春风。

          # 大风歌 刘邦

          大风起兮云飞扬， 威加海内兮归故乡， 安得猛士兮守四方！

          # 山居秋暝 王维

          空山新雨后，天气晚来秋。
          明月松间照，清泉石上流。
          竹喧归浣女，莲动下渔舟。
          随意春芳歇，王孙自可留。

          # 桃花庵歌 唐寅

          桃花坞里桃花庵，桃花庵里桃花仙。
          桃花仙人种桃树，又摘桃花卖酒钱。
          酒醒只在花前坐，酒醉还来花下眠。
          半醒半醉日复日，花落花开年复年。
          但愿老死花酒间，不愿鞠躬车马前。
          车尘马足富者趣，酒盏花枝贫者缘。
          若将富贵比贫贱，一在平地一在天。
          若将贫贱比车马，他得驱驰我得闲。
          别人笑我太疯癫，我笑他人看不穿。
          不见五陵豪杰墓，无花无酒锄作田。
